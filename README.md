
# Projet Examen DevNet - Monitoring Réseau

Application Flask de monitoring réseau avec API REST, conteneurisée avec Docker et utilisant PostgreSQL.

## 🚀 Fonctionnalités
- Interface web pour visualiser les équipements réseau
- API REST complète pour accéder aux données
- Statistiques en temps réel
- Base de données PostgreSQL
- Conteneurisation avec Docker

## 🛠 Technologies utilisées
- **Flask** : Framework web Python
- **PostgreSQL** : Base de données
- **Docker** : Conteneurisation
- **Docker Compose** : Orchestration
- **GitHub** : Versionnement du code
- **Docker Hub** : Publication de l'image

## 📦 Installation

### Prérequis
- Docker et Docker Compose installés
- Git (optionnel)

### Avec Docker Compose
```bash
# Cloner le projet
git clone https://github.com/minahtaw44-beep/projet-examen-devnet.git
cd projet-examen-devnet

# Lancer l'application
docker-compose up
